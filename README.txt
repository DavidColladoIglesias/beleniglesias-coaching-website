# Belén Iglesias — Coach Profesional

**[🔗 Ver el proyecto en vivo](https://beleniglesias.es)**

Sitio web completo para un negocio de coaching y mindfulness, construido de cero: desde la arquitectura de la web y el diseño, hasta el sistema de reservas, la tienda online y la infraestructura de seguridad y SEO.

## 🎯 Sobre el proyecto

Web profesional para una coach personal, pensada para que sus clientes puedan:
- Descubrir sus servicios (sesiones individuales y planes de varias sesiones)
- Reservar sesiones de coaching online directamente desde la web, con calendario de disponibilidad en tiempo real
- Comprar su libro publicado, tanto en formato digital como físico
- Contactar y suscribirse a novedades

## 🛠️ Stack técnico

- **CMS:** WordPress
- **Maquetación:** Elementor + Elementor Pro / PRO Elements
- **Tienda / Pagos:** WooCommerce, Stripe, PayPal
- **Reservas:** MotoPress Appointment Booking, WooCommerce Bookings
- **SEO:** Rank Math (schema markup, sitemap, SEO local, Search Console)
- **Seguridad:** Wordfence (firewall, escaneo de malware, autenticación)
- **Backups:** UpdraftPlus, All-in-One WP Migration
- **Automatización / integraciones:** WPCode (snippets PHP personalizados), Bit Integrations
- **Marketing:** WebToffee (email marketing, popups), WP Mail SMTP
- **Cumplimiento legal:** CookieYes (RGPD/cookies), UserWay (accesibilidad WCAG/ADA)

## ✨ Funcionalidades desarrolladas

- **Sistema de reservas de sesiones de coaching**, con calendario de disponibilidad configurado por horarios, ubicación y duración de bloques
- **Tienda online (WooCommerce)** con productos de sesión única y packs de varias sesiones (con precios escalonados)
- **Producto digital descargable** (libro en formato ePub) con gestión de archivos y límites de descarga
- **Personalización con código a medida** (snippets PHP vía WPCode) para adaptar comportamientos que los plugins no cubrían de forma nativa
- **Configuración de seguridad** (firewall, protección de login, escaneo de malware)
- **Optimización SEO** (schema de datos estructurados, sitemap, SEO local, analítica conectada a Search Console)
- **Sistema de copias de seguridad** automatizado y verificado
- **Cumplimiento RGPD** (gestión de cookies y consentimiento)
